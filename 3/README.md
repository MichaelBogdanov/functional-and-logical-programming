Сравним эффективность двух методов сортировки: сортировка простыми вставками имеет квадратичную сложность (O(n^2)), поскольку в худшем случае для каждого элемента потребуется пройти по всей отсортированной части массива, из-за этого неэффективна для больших наборов данных, а сортировка Шелла основана на улучшении сортировка вставками, путем использования большого шага для сравнения и перестановки элементов, в случае с моим вариантом для генерации последовательности шагов использовался метод Седжвика при котором, согласно [Википедии](https://ru.wikipedia.org/wiki/Сортировка_Шелла), средняя алгоритмическая сложность будет O(n^(7/6)), а в худшем случае O(n^(4/3)), так что можно смело сказать, что сортировка Шелла в этом плане будет эффективнее
